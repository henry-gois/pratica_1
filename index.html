Praticando com o Git e Github

O que é o GIT?

Git é um repositório que podemos ligar com o VScode para que salve um "histórico" para coisas que realizarmos dentro Vs. Este é muito funcional e pode ajudar em diversas situações, além de poder ser enviado para o GitHub, um repositório

O que é o Github?

GitHub é uma rede social para Dev's que pode ser utilizado como repositório remoto ao ser usado com o Git. É de simples uso e ajuda muito a vida de programadores para trabalhar em grupo.
